# Presentation API

Простое REST API на Node.js для получения тестовых данных.

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/macrulezru/presentation-api.git
cd presentation-api

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev
```

## Что умеет

- Получать список всех продуктов
- Получать конкретный продукт по ID
- Получать случайный продукт

- Получать список всех пользователей
- Получать пользователя по ID
- Получать случайного пользователя

## Использование

API доступен по следующим эндпоинтам:

- `GET /api/products` – получить все продукты
- `GET /api/products/:id` – получить продукт по ID
- `GET /api/randomProduct` – получить случайный продукт

- `GET /api/persons` – получить всех пользователей
- `GET /api/persons/:id` – получить пользователя по ID
- `GET /api/randomPerson` – получить случайного пользователя
